# SANDHU-SECURE
Modern OTP Authentication System using HTML, CSS, JavaScript and Flask with secure verification, timer, login/register UI and responsive design.

Live Demo:
https://sandhu-secure.onrender.com

📌 About The Project

SANDHU SECURE is a modern OTP authentication system designed with a professional responsive UI and secure verification flow.

This project was built to practice:

Frontend Development
Backend Integration
Authentication Systems
Flask Deployment
Responsive UI/UX Design
OTP Verification Logic

The project includes features like:

OTP Generation
OTP Verification
Countdown Timer
Password Validation
Show/Hide Password
Caps Lock Detection
Login & Register System
Responsive Modern Design
WhatsApp / SMS / Email OTP Selection UI
